# Task Management API with MongoDB

A RESTful Task Management API built with **Node.js**, **Express.js**, and **MongoDB (Mongoose)**.  
Features include **JWT authentication**, **password hashing**, **task CRUD with user association**, and **pagination/filtering**.  
The API is production-ready and deployable on **Render/Railway** with **MongoDB Atlas**.
